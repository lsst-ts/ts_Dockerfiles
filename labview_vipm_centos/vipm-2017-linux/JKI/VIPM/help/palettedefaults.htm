Palette Defaults

Figure 50 - Palette Defaults Section

Apply Short Names to VI Titles: This option defines the default behavior for how VIPM manages VI Titles for VIs in the palettes.

If this option is not checked (default setting), VIPM does the following:

•      Does not modify the VI Titles of the palette VIs

•      It will modify the VI Title if you have set one on the individual palette item by right-clicking on the palette item and selecting “Edit VI Title”. In other words, it repects any individual customizations you have made to the item in the palettes.

If this option is checked, VIPM does the following:

•      Does modify the VI Titles of the palette VIs to use short names. The default short name is the VI name (without prefix or suffix) and without the extension (.VI).

•      It will modify the VI Title if you have set one on the individual palette item by right-clicking on the palette item and selecting “Edit VI Title”. In other words, it respects any individual customizations you have made to the item in the palettes.

•      It will use the short name you have defined for the individual item. In other words, if you have set a custom short name for the palette item, VIPM will use that instead of the default short name.
