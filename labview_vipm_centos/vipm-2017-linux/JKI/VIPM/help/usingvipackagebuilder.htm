Using VI Package Builder

Also Visit this link for a guide on getting started with building VI Packages.

See: Debugging Package Build Errors for more help with problems building your packages.

The VI Package Builder window allows you to build a VI Package from a folder of reusable VIs that can be installed into the LabVIEW palettes and other areas of the LabVIEW development environment. This window can be launched from the main VIPM window by:

•      Selecting Tools>Build Package.

•      Clicking on the button on the main VIPM window.
