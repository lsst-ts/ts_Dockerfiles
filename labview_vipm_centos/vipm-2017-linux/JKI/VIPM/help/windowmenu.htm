Window Menu

Figure 13 - Window Menu

Show VI Package Configuration Editor: This will show the VI Package Configuration Editor window where you can edit and create package configurations.

Show VI Package Builder: This will show the VI Package Builder window where you can build packages out of your reuse libraries.

Show VI Package Repository Manager: This will show the VI Package Repository Manager Window where you can add and remove packages from your repository.

Show LabVIEW Tools Network: This will show the LabVIEW Tools Network View where you can browse and install products from the LabVIEW Tools Network.
