View Menu

Figure 9 - View Menu

Show:

All: Show all packages in the package list regardless of group type.

Installed: Only show packages in the package list that are currently installed

Not Installed: Only show packages in the package list that are not installed.

Missing Dependency: Only show packages in the list that have missing dependencies or other errors.

Upgrades: Only show installed packages in the package list that have newer versions available.

Legend: This item can toggle on (state indicated by a checkmark) the VIPM legend. If the legend is already visible, then this menu item will turn the legend off. The legend is displayed in the bottom right corner of the package list.

Figure 10 - VIPM Legend

Error List: This will display the VIPM Error List dialog which lists any problems with the current installed packages.

Note: This menu item is only enabled if there are problems.
