Package Builder Options – VI Package Builder

This section configures options pertaining to the VI Package Builder window.

Figure 98 - VI Package Builder options page

Warn me before updating icons on package, organization, or submenu change: If this option is selected, you will be presented with the following dialog, after you edit a package name, organization name, or submenu name.  If you choose “Remember my decision” in the dialog, the option will be unchecked – you can reset your choice by re-selecting the option.

Warn me before launching LabVIEW when building a package: If this option is selected, you will be presented with the following dialog, when you build a package.  If you choose “Remember my decision” in the dialog, the option will be unchecked – you can reset your choice by re-selecting the option.

Warn me before updating functions and controls palette icons with the package icon: If this option is selected, you will be presented with the following dialog, after you edit the package icon.  If you choose “Remember my decision” in the dialog, the option will be unchecked – you can reset your choice by re-selecting the option.
