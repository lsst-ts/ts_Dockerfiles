VI Package Manager (VIPM) > Installing VIPM

System Requirements

Internet Network Connection

VIPM relies on the internet to get package updates on the LabVIEW Tools Network and the VI Package Network. A high speed internet connection is recommended. VIPM also provides configuration parameters to setup a proxy server connection. An Internet connection may also be required if you are connecting to a corporate repository that is located remotely.

Note: You do not need an internet network connection if you are not performing the above actions.

Hard Disk Drive Storage Space

The VIPM installation requires approximately 100MB of disk space. Depending on the number of packages you plan on installing, VIPM will also require the appropriate disk storage above the initial install requirement. This is package dependant.

Supported LabVIEW Versions

VIPM is able to manage your reuse libraries (install\uninstall) for all LabVIEW versions starting at version 7.1 and up to the most recent version. VIPM will not work with LabVIEW versions older than LabVIEW 7.1.

Also, VIPM can build packages from LabVIEW 8.2 and newer. VIPM cannot build packages in versions of LabVIEW older than 8.2. See here for more information on building packages with VIPM.
