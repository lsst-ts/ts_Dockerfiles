Add Palette to Library or Class

If you enable this then VIPM will set the current palette as the default for an LVLIB or LVCLASS file in your project. What does this mean?

LabVIEW Classes: When you right-click on a class wire, you will get a menu to select VIs from the palette created in VIPM.

LVLIBS: When you right-click on a VI who’s a member of the library, you will get a menu to select VIs from the palette created in VIPM.
