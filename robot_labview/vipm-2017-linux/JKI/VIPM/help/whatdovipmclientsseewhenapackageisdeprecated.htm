What do VIPM Clients See When a Package is deprecated?

VIPM clients have the option to show or hide deprecated packages from their main package list display. This is configurable in the VIPM General Options. Installed packages that are also deprecated will continue to show up regardless of that setting. However, once a deprecated package is uninstalled and they have chosen to hide deprecated packages, then they will no longer see this package.

VIPM also indicates deprecated packages in several places on the user interface.
